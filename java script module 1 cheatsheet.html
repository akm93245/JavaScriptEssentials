JavaScript Development Cheatsheet

Tag/Terminology Description Code Example
<script> Used to include the required JavaScript code in your HTML document. 
<body>
    <p id="showname"></p>
    <script>
        document.getElementById('showname')
        .innerHTML='Peter';
    </script>
</body>
<script src> Used to link the required JavaScript files in your HTML document. 
<script src="script.js"></script>
var var is a keyword used to declare variables. 
var num1=10;
var num2=11;
var & Scope var has functional scope, allowing variable to be accessed within function only. 
function show() {
    var name = 'Peter';
    document.getElementById('showname')
    .innerHTML = name;
}
let let is a keyword used to declare variables. 
let num1=20;
let num2=21;
let & Scope let has block scope, allowing the variable to be limited to the block, statement, or expression in which it is defined, preventing redeclaration within the same scope. 
{
    let emailId = 'test@example.com';
    document.getElementById('showemail')
    .innerHTML = emailId;
}
const const is a keyword used to declare variables. 
const employeeId=120;
const employeeId=121;
const & Scope It creates a constant whose value cannot be reassigned or redeclared. 
{
    const employeeId = 120;
    document.getElementById('showeEId')
    .innerHTML = employeeId;
}
Arithmetic Operators Arithmetic operators perform mathematical calculations like addition, subtraction, multiplication, division and modulus. 
let x = 15, y = 3;
let sum = x + y; // 18
let difference = x - y; // 12
let product = x * y; // 45
let quotient = x / y; // 5
let remainder = x % y; // 0
Comparison Operators Comparison operators compare values and return true/false based on the comparison. 
let a = 5, b = 7;
let isEqual = a == b; // false
let isNotEqual = a != b; // true
let isStrictEqual = a === b; // false
let isGreaterThan = a > b; // false
Logical Operators Logical operators combine multiple conditions and return a boolean result. 
let hasPermission = true;
let isMember = false;
let canAccess = hasPermission && isMember;
let canView = hasPermission || isMember;
let isDenied = !hasPermission;
Assignment Operators Assignment operators assign values to variables. For example, =, +=, -=. 
let x = 10;
x += 5; // x = x + 5
x -= 5; // x = x - 5
Unary Operators Unary operators act on a single operand, performing operations like negation or incrementing. 
let count = 5;
count++; // count is now 6
count--; // count is now 5 again
typeof Operator typeof operator returns the data type of a variable or expression as a string. 
let num1 = 42;
console.log(typeof(num1)); // Number
let name = 'John';
console.log(typeof(name)); // String
if Statement The if statement is used to execute a piece of block code if the given condition is true. 
let age = 25;
if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}
else if Statement It allows you to test multiple conditions sequentially. If the condition is true then it will execute if statement block otherwise execute else statement block. 
let month = 'March to May';
if (month == 'March to May') {
    console.log('It is spring season');
} else if (month == 'June to August') {
    console.log('It is summer season');
} else if (month == 'Sep to Nov') {
    console.log('It is autumn season');
} else {
    console.log('It is winter season');
}
Nested if else This statement allows you to test multiple conditions and execute different blocks of code based on the results of those conditions. 
const temp = 30, isRaining = true;
if (temp > 30) {
  if (isRaining) {
    console.log("Hot and raining.");
  } else {
    console.log("Hot, not raining.");
  }
} else {
  if (isRaining) {
    console.log("Not hot, raining.");
  } else {
    console.log("Nice day.");
  }
}
switch Statement The switch statement is used for multiple conditional branches, allowing the execution of different code blocks based on the value of an expression. 
let month = "December";
switch (month) {
    case "December":
        console.log("Christmas month.");
        break;
    case "November":
        console.log("Thanksgiving month");
        break;
    default:
        console.log("Regular month.");
}
Ternary Operator The ternary operator is the simplest way to write conditional statements such as if else condition. 
let age = 20;
let canVote = age >= 18 ? "Yes" : "No";
for loop A for loop is a control structure that allows to execute a block of code repeatedly for a specified number of times until a particular condition is met. 
for (let i = 1; i <= 5; i++) {
    console.log(i);
}
while loop A while loop is a control structure that allows to execute a block of code repeatedly as long as a specified condition is true. 
let limit = 50, a = 0, b = 1;
while (a <= limit) {
    console.log(a);
    let temp = a + b;
    a = b;
    b = temp;
}
do while loop A "do...while" loop allows you to execute a block of code repeatedly as long as a specified condition is true and guarantees that the code block will execute at least once. 
let roll = 1;
do {
    console.log("Rolled a " + roll);
    roll++;
} while (roll < 7);
Function Declaration Function is a reusable block of code that can be defined and executed as many times as needed. 
function sayHello() {
  console.log("Hello!");
}
sayHello(); // function call
Non-Parameterized Functions The functions that do not require any parameters to operate. 
function greet() {
  const greeting = "Hello, World!";
  console.log(greeting);
}
greet();
Parameterized Functions The function that accepts one or more values that provide input data for the function to work with. These values are called parameters in declaration and arguments when calling. 
function add(a, b) {
    return a + b;
}
console.log(add(3, 4)); // 7
Named Function The functions with a specific name that can be called by that name. 
const add = function(a, b) { 
    console.log(a+b); 
}
add(2, 3); // 5
IIFE Immediately Invoked Function Expression is a function in JavaScript that's defined and executed immediately after its creation. 
(function sayWelcome() {
  console.log("Welcome!");
})();
Arrow Function Arrow functions in JavaScript are a concise way to write function expressions, using the => syntax. 
const arrowFunc = (a, b) => a + b;
console.log(arrowFunc(5, 3)); // 8
return The return statement in JavaScript is used to end the execution of a function and specify the value that the function should return to the caller. 
function multiply(a, b) {
    return a * b;
}
let result = multiply(3, 4); // 12
Function Closure A function closure in JavaScript allows a function to access and remember variables from its outer scope even after that scope has finished executing. 
function outerFunction() {
  const outerVar = "I am outer";
  function innerFunction() {
    console.log(outerVar);
  }
  return innerFunction;
}
const closure = outerFunction();
closure(); // "I am outer"
Function Hoisting Function hoisting means that function declarations are moved to the top of their containing scope during the compile phase, allowing them to be used before they are declared in the code. 
sayHello(); // Works!
function sayHello() {
  console.log("Hello!");
}
Function Hoisting (Expression) Function expressions where a function is assigned to a variable do not exhibit hoisting behaviour. 
greet(); // Error!
const greet = function() {
  console.log("Greetings!");
};
addEventListener addEventListener is a JavaScript method used to assign a function to execute when a specific event occurs on an element in the DOM. 
const button = 
  document.getElementById('btn');
button.addEventListener('click', () => {
  console.log('Button clicked!');
});
onclick Event A way of assigning a function directly to an HTML element to execute when it's clicked. 
<button onclick="myFunction()">
  Click me
</button>
<script>
function myFunction() {
  alert('Button clicked!');
}
</script>
mouseover Event The mouseover event is triggered when the mouse cursor enters an element. 
const myDiv = 
  document.getElementById('myDiv');
myDiv.addEventListener('mouseover', () => {
  myDiv.style.backgroundColor = 
    'lightgreen';
});
mouseout Event The mouseout event in JavaScript is triggered when the mouse pointer moves out of an element. 
const myDiv = 
  document.getElementById('myDiv');
myDiv.addEventListener('mouseout', () => {
  myDiv.style.backgroundColor = 
    'lightcoral';
});
keydown Event The keydown event is triggered when a key on the keyboard is pressed down. 
const input = 
  document.getElementById("myInput");
input.onkeydown = function(event) {
  console.log('Key: ' + event.key);
};
change Event The change event is triggered when the value of an input element changes. Typically used for form elements like text fields or dropdowns. 
const input = 
  document.getElementById("myInput");
input.onchange = function() {
  console.log('Value: ' + input.value);
};
onsubmit Event The onsubmit event in HTML occurs when a form is submitted, either by clicking a submit button or by calling submit(). 
<form onsubmit="validateForm()">
  <input type="text" id="name">
  <input type="submit">
</form>
<script>
function validateForm() {
  event.preventDefault();
  // validation logic
}
</script>
